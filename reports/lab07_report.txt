How is an image converted into numbers?
The image is resized to 64x64, normalized to a 0.0-1.0 range, and flattened into a 12,288-dimensional vector (64x64x3 channels)

What does the model actually "see"?
The model sees a long sequence of numerical values representing pixel intensities and color patterns in a high-dimensional space

Why can the model classify random or unknown images as one of the known classes?
The model is restricted to a closed-world output; it must assign the input to the class with the highest mathematical probability among its known labels (Forest, River, or Residential)

Why can this behavior be dangerous in real AI systems?
It leads to overconfident incorrect predictions, which can be critical in satellite missions for navigation or disaster management.

What would need to be improved before using such a system operationally?
Implementing a confidence threshold, adding an unknown/other class, and using more robust architectures like Convolutional Neural Networks (CNN)

Which model achieved the highest accuracy?
Random Forest or SVM usually achieve the highest accuracy for this specific feature extraction method.

Which model was fastest to train?
KNN or Logistic Regression are generally the fastest to train

Did the fastest model also achieve the best accuracy?
Usually no; there is a trade-off where simpler, faster models often lack the capacity to capture complex visual patterns compared to slower ensemble models.

Did all models classify the same image in the same way?
Not always; different algorithms interpret the feature vector differently, leading to varying predictions on ambiguous images.

What does the accuracy vs training time plot show?
It visualizes model efficiency, helping to identify which algorithm provides the best performance relative to its computational cost.
